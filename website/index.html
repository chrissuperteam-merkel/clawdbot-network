How it works

Four steps. Real traffic. Real payments.

No middlemen. No subscriptions. Agents pay nodes directly through Solana smart contracts.

01

Phone Connects

Install the Clawdbot app. Your phone connects to the network as a proxy node via WebSocket.

02

Agent Pays Escrow

Agent sends 0.005 SOL to the escrow contract. Payment is verified on-chain before the session starts.

03

Traffic Routes

HTTP and HTTPS traffic flows through the phone's real IP. Full tunnel — TLS, HTTP/2, the works.

04

Node Gets Paid

Session ends. 70% goes to the phone owner, 30% platform fee. Instant. On-chain. Transparent.

For Developers & Agents

Three lines. Real residential IP.

No SDK to install. No authentication dance. Just HTTP.

// Quick test — one line
curl "/proxy/fetch?url=https://httpbin.org/ip"
// → { "origin": "87.191.59.123" }  ← phone's real IP

// Full flow with Solana payment
const session = await fetch('/proxy/session', {
  method: 'POST',
  body: JSON.stringify({
    wallet: 'YourAgentWallet...',
    escrowTx: 'SolanaTransactionSig...',
    country: 'DE'
  })
})

// Use as HTTP/HTTPS proxy
curl --proxy host:1080 \
  -H "X-Session-Id: ${session.id}" \
  https://any-website.com

API Reference

Simple REST API. No SDK required.

GET /nodes

List all online proxy nodes with device info, country, and connection type.

POST /proxy/session

Create a proxy session. Optionally verify Solana escrow payment on-chain.

GET /proxy/fetch?url=

One-liner proxy test. Send any URL, get the response through a phone's IP.

POST /proxy/session/:id/end

End session and trigger automatic SOL payout to the node owner.

TCP Proxy :1080

Full HTTP/HTTPS CONNECT proxy. Use with curl, Python requests, or any HTTP client.

GET /admin/health

Network health check. Nodes online, active sessions, uptime, Solana network status.
